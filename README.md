# Python Password Manager

## Description
A small Python project that generates passwords and stores account credentials in a `.json` file.
The project is intended for practicing file handling and basic data management in Python.

## Features
- Generate random passwords
- Save account login and password to a JSON file
- Read stored credentials from the file

## Installation
Clone the repository:
```bash
git clone https://github.com/Bo-Br/Python-Password-Generator.git

```
Then, go to the main project folder using :
```bash
cd Python-Password-Generator
```
And execute the code by using :
```bash
python main.py
```
